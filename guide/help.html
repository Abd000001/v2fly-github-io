# 寻求帮助

Project V 提供了多种方式进行交流。

TIP

Project V 团队支持中文和英文，为避免一些不必要的误会，请选择你所熟悉的语言来提问。管理员会以问题发起者使用的语言来回复；如若提问者使用了其它的语言，则以英文回复。

# Telegram 讨论组

Project V 提供了下列讨论组，用于讨论不同类型的问题。

  • 使用 (opens new window)

    • 话题限定
    • 只允许发送特定表情

  • 日常 (opens new window)

    • 用于其它不相关的话题。

WARNING

所有讨论组均不可发布 NSFW、政治相关、仇恨言论、阴谋论、传播（疑似）谣言、蓄（恶）意冒充他人和广告、策划违法活动等。

另外请关注 V2Fly 公告频道 (opens new window) 以获得最新资讯。

# Email

如果你想和 V2Fly 的开发人员私下讨论一些问题，请通过下列的邮箱联系。

dev@v2fly.org: 主要通信邮箱，基本每天都查看。 security@v2fly.org: 仅用于沟通涉及安全隐患的内容。

TIP

由于工作繁忙，不能保证每封邮件都回复，请见谅。请尽量使用社区讨论 (opens new window)以得到更快速的响应。
