        # 第三方图形客户端

        Qv2ray

        跨平台 V2Ray 客户端，支持 Linux、Windows、macOS，可通过插件系统支持 SSR / Trojan / Trojan-Go / NaiveProxy 等协议

        SagerNet

        SagerNet 是一个基于 V2Ray 的 Android 通用代理应用。

        V2rayN

        V2RayN 是一个基于 V2Ray 内核的 Windows 客户端。

        v2rayA

        基于 web GUI 的 Linux 客户端，支持全局透明代理，兼容SS、SSR、Trojan、PingTunnel协议。

        # 路由规则

        V2Ray Rules Dat

        V2Ray 路由规则文件加强版，可代替官方 geoip.dat 和 geosite.dat @Loyalsoldier/v2ray-rules-dat (opens new window)
